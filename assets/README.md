# Assets

These are placeholder PNGs (currently copied from wandercoin) so that
`expo start` and EAS builds succeed during development. Replace them
before publishing:

- `icon.png` — 1024×1024, app icon.
- `adaptive-icon.png` — 1024×1024 foreground for Android adaptive icon.
- `splash-icon.png` — 1024×1024 splash image.
- `favicon.png` — 48×48 web favicon.
